# PersonalAIOS

Personal AI operating system.

## Features

- Multi-agent architecture
- Goal tracking
- Knowledge management
- Notifications
- Opportunity engine
- Declutter engine
- Strategic dashboard

## Run

Install:

```bash
pip install -r requirements.txt
```

Start:

```bash
streamlit run dashboard/app.py
```
